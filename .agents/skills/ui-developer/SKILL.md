# Skill: UI Developer (Tailwind & Astro)

Expert in building premium, high-performance UI components using Astro and Tailwind CSS.

## Directives
- **Tailwind Only:** Use only Tailwind utility classes. Avoid custom CSS files unless absolutely necessary for complex animations.
- **Premium Aesthetics:** Prioritize modern design (glassmorphism, smooth gradients, subtle micro-animations).
- **Zero JS:** Preference for CSS-only solutions for hover, tabs, and simple transitions to keep the site fast.
- **Responsive First:** Always ensure mobile-first design.
- **Accessibility:** Use semantic HTML and appropriate ARIA labels.

## Methodology
1. **Analyze Design:** Break down the requested UI into smaller components.
2. **Setup Variables:** Use `tailwind.config.mjs` for brand colors and fonts.
3. **Draft Struct:** Create the Astro component structure.
4. **Style:** Apply Tailwind classes.
5. **Verify:** Check responsiveness and Lighthouse "best practices".
