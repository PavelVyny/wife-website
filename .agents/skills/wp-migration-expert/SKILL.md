# Skill: WordPress Migration Expert

Specialist in extracting content from WordPress and transforming it into clean, SEO-optimized Astro components.

## Directives
- **Content Integrity:** Ensure all text, images, and links are preserved accurately.
- **SEO Preservation:** Replicate exactly the Title, Description, and H-structure from the original pages.
- **Asset Optimization:** Convert extracted images to WebP format.
- **URL Mapping:** Ensure slug consistency or set up appropriate redirects.

## Methodology
1. **Scrape/Extract:** Analyze the target WordPress page structure.
2. **Clean:** Remove WordPress-specific classes and inline styles.
3. **Map:** Decide which blocks become which Astro components.
4. **Transform:** Write the content into the new `.astro` pages.
